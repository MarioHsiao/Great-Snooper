# The text of the popup message when About WormNat2 is clicked
AboutWormNat2Text=Über WormNat 2: Wenn du über das Spiel hostest, wird auf deinem Computer der Server erstellt, und Spieler verbinden sich über einen bestimmten Port (normalerweise 17011). Die meisten Router und Firewalls erlauben keine eingehenden Verbindungen, so dass niemand dem Spiel beitreten kann. Damit Spieler dem Spiel beitreten können müssen in der Router- oder der Firewall-Konfiguration Ports geöffnet werden. WormNat2 nutzt eine Technik, die erlaubt Spiele zu hosten ohne eingehende Verbindungen zu erlauben. Um dies zu ermöglichen benutzt es einen dedizierten Proxy-Server, welcher alle Verbindungen handhabt. Da keine anderen Proxy-Server unterstützt werden, wird allerdings empfohlen, das System so einzurichten, dass das Hosten von Spielen auch ohne WormNat 2 möglich ist, und WormNat nur benutzt wird, wenn es gar nicht anders möglich ist, damit der Proxy-Server nicht überlastet wird. Es ist nicht notwendig, Wormkit Module im Spiel zu aktivieren, um WormNat 2 über den Snooper zu verwenden!
